## Project Title
Smart Habit Tracker

## Problem Statement
Many students struggle to build consistent daily habits. This tracker provides a simple
console-based tool to record habits and view progress.

## Scope
- Single-user, local (file-based) tracker.
- Tracks completion dates and simple streaks.
- No network or database required.

## Target Users
Students and learners who want a lightweight tool to track daily habits.

## High-level Features
- Add/Edit/Delete habits
- Mark daily completion
- Display total completions and current streak
- Save/load data to/from file
